## Selected Projects

**VAIKUNTH — Satellite Cloud Removal Pipeline**
*ISRO Bharatiya Antariksh Hackathon 2026, Team CloudClear AI*
Cloud-removal pipeline for LISS-IV satellite imagery combining U-Net, Pix2Pix GAN/latent-diffusion inpainting, and Sentinel-1 SAR fusion. Delivered as a complete submission with technical documentation for ISRO evaluation.

**Energy-Constrained Latent World Dynamics Model (ECL-WDM)**
Physics-informed model (TCN + Gated Neural ODE + energy-based causal regularization) for continuous-time dynamics, achieving stable long-horizon predictions in chaotic systems. Published, IEEE CSNT 2026.

**Synaptiq RAG — Advanced Retrieval-Augmented Generation System**
High-performance RAG system outperforming HippoRAG on retrieval accuracy and multi-hop reasoning benchmarks, with optimized latency for domain-specific knowledge tasks.

**MushVision / MushVision PRO**
Shipped iOS app (LiDAR + CV) for mushroom farm bed scanning and before/after harvest comparison, live on the App Store. Currently building MushVision PRO, a standalone NVIDIA Jetson edge device for real-time on-device detection.
